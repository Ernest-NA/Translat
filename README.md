# Translat
Translat es una aplicación de escritorio para Windows 11 que orquesta traducciones del inglés al castellano con IA. Gestiona proyectos, glosarios, perfiles de estilo, reglas editoriales, corpus paralelos EN/ES, búsqueda de traducciones específicas, control de costes, trazabilidad y aprendizaje supervisado a partir de la edición humana.
